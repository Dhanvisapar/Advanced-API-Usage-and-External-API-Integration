# Advanced-API-Usage-and-External-API-Integration

## Objective
To integrate an external API and implement advanced API concepts.

## Features
- External API integration (OpenWeather)
- API authentication using API key
- Rate limiting
- Error handling
- Client-server communication

## How to Run
1. Open folder in VS Code
2. Run:
   npm install
   npm start
3. Open browser:
   http://localhost:3000

## Learning Outcomes
- External API usage
- API security concepts
- Rate limiting
- Error handling
